# Python & cadwork 3d API

Welcome to the lecture material for **Python programming** and the **cadwork 3d Python API** at Berner Fachhochschule.

## Course Overview

This course is structured into two main parts:

1. **Python Basics** — Core Python concepts including data types, control flow, functions, and object-oriented programming.
2. **cadwork 3d API** — Applying Python within cadwork 3d to automate timber construction workflows.

## Getting Started

Use the navigation tabs above to explore the lecture material. Each section builds on the previous one, so it is recommended to follow them in order.

!!! tip "Prerequisites"
    No prior programming experience is required. All concepts are introduced from the ground up.
