# Python Basics

This section is a **refresher** on Python fundamentals. Each topic mixes short explanations, code examples, and **Quick Check** questions you can answer before revealing the solution.

## How to use this section

Each page contains blocks like this:

!!! question "Quick Check"
    Try to answer first, then click the toggle below.

    ??? success "Show answer"
        Answers appear here when you click.

Work through them in order — concepts build on each other.
